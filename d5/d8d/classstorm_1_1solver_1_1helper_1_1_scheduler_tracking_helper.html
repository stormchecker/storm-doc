Computes the optimal choices from the given solution.

Essentially, this applies one iteration of value iteration and stores the optimal choice for each state.

Parameters
operandInOperand for the value iteration step. Should hold the solution values of the MinMax equation system
offsetsOffsets that are added to each choice result.
dirOptimization direction to consider.
schedulerStoragewhere the scheduler choices will be stored. Should have the same size as the operand(s).
uncertaintyResolutionModeRepresents the mode indicating how the uncertainty should be resolved.
operandOutif given, the result values of the performed value iteration step will be stored in this vector. Can be the same as operandIn.
Returns
True if the scheduler coincides with the provided scheduler encoded in schedulerStorage
Note
: schedulers are encoded using row indices that are local to their row group, i.e. schedulerStorage[i]==j means that we choose the j'th row of row group i
