Detailed Description

template<typename ValueType = double>
struct storm::parser::MarkovAutomatonSparseTransitionParser< ValueType >::FirstPassResult

A structure representing the result of the first pass of this parser.

It contains the number of non-zero entries in the model, the highest state index and the total number of choices.
